# SkyCast Weather App

SkyCast Weather App is a simple React.js weather application.

## Features

- React component structure
- React hook: useState
- API fetching
- Loading and error messages
- Responsive design
- Simple clean UI

## Available Cities

- Los Angeles
- New York
- London
- Dhaka
- Tokyo

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project can be deployed on Vercel, Netlify, or GitHub Pages.
